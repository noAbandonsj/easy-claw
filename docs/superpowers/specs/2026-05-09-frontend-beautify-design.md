# Frontend Beautify Design

## 目标

对 easy-claw 的 Web 聊天前端进行全面视觉升级，从单页布局改为双栏布局，采用赛博霓虹配色方案，提升整体视觉品质。

## 设计决策

| 维度 | 选择 |
|------|------|
| **配色** | 赛博霓虹 — 深蓝黑底 (`#0a0a1a`/`#0d0d25`) + 粉色 (`#ff69b4`) + 青色 (`#00ffff`/`#66ffff`) |
| **风格** | 平衡质感 — 玻璃态面板、渐变边框、霓虹发光、消息气泡光晕 |
| **布局** | 双栏 — 左侧 220px 会话侧栏 + 右侧聊天区 |
| **左侧栏** | 新建对话按钮 + 会话列表(标题/模型/时间) + 底部状态(模型/工作区/Token) |
| **保留** | 所有现有斜杠命令、WebSocket 协议、Markdown 导出、Modal 弹窗系统 |

## 组件清单

### 1. 左侧栏 (Sidebar)

- **Logo 区**: `◆ easy-claw` + 版本号标签(青底)
- **新对话按钮**: 粉渐变背景 + 霓虹发光 box-shadow，点击 POST `/sessions` 创建新会话
- **会话列表**: 从 GET `/sessions` 获取，显示标题、模型名、相对时间。当前激活项带青边框高亮
- **底部状态**: 实时显示当前模型、工作区路径、Token 累计用量

### 2. 顶栏 (TopBar)

- 连接状态指示灯（青点 + 发光）
- 当前会话标题
- 右侧状态文字（就绪/思考中/错误）

### 3. 消息区

- **用户气泡**: 粉渐变背景、右对齐、右下角直角、微弱粉色发光
- **助手气泡**: 玻璃态面板(半透明背景)、左对齐、左下角直角、边框光晕
- **流式打字光标**: 粉色竖线，闪烁动画
- **工具面板**: 青色半透明边框、折叠展开、monospace 内容区

### 4. 输入区

- 文本框: 深色底 + 粉色边框(focus 时发光增强)
- 发送按钮: 粉渐变 + 霓虹 glow box-shadow
- disabled 状态: 边框变灰、按钮去色

### 5. Modal (不变)

保持现有弹窗结构，配色适配新方案：表面色、青色表头文字

## 交互行为

- **会话切换**: 点击左侧会话 → 断开当前 WS → 用新 session_id 重连 → 清空消息区
- **新对话**: 点击按钮 → POST `/sessions` → 切换到新会话
- **斜杠命令**: `/help`、`/status`、`/skills`、`/mcp`、`/browser`、`/sessions` 等全部保留
- **断线重连**: 保持现有 2 秒自动重连逻辑
- **键盘**: Enter 发送、Shift+Enter 换行、Escape 关闭弹窗

## 不变内容

- WebSocket 协议 `/ws/chat` — token/tool_call_start/tool_call_result/done/banner/error 事件格式不变
- HTTP API — 所有端点路径和响应格式不变
- 斜杠命令分发逻辑 — `handleSlash()` 函数保留
- Markdown 导出 — `downloadMarkdown()` 保留
- 后端代码 — 零改动

## 技术约束

- 单文件 `index.html`，无框架，无构建步骤
- inline CSS + vanilla JS
- 桌面端优先，不要求移动端响应式
- 兼容 Chrome/Edge/Firefox 现代版本

## CSS 特效清单

| 特效 | 应用位置 | 实现方式 |
|------|----------|----------|
| 霓虹发光 | 发送按钮、连接指示灯 | `box-shadow` 多层叠加 |
| 玻璃态面板 | 助手气泡、工具面板 | `background: rgba(18,18,42,0.8)` + `backdrop-filter: blur()` |
| 渐变边框 | 输入框 focus、激活会话 | `border-image` 或伪元素渐变 |
| 渐变背景 | 用户气泡、新对话按钮、发送按钮 | `linear-gradient` |
| 脉冲光晕 | 工具面板边框、连接点 | `@keyframes` 动画 |
| 平滑过渡 | hover、focus、切换 | `transition` |

- 不改变后端 Python 代码
